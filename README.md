# Seeker of the Sword
An ASCII-character based adventure world written in python.
(Executes into console.)

### MOVEMENT  
`w` - forward  
`a` - left  
`d` - right  
`s` - back  

### MISCELLANEOUS  
`z` - exit program  
`e` - enter / exit inventory  

### ENVIRONMENT   
`M` - mountain  
`^` - forest  
`o` - lake / river (Windows)  
`.` - lake / river (OS X)  
`O` - origin  

### TO START GAME  
`python adventure.py`
